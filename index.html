Interactive Board Game Recommendation Map (40,000+ BGG Games)

This interactive board game recommendation tool visualizes relationships between more than 40,000 titles from BoardGameGeek. Each node represents a game, and distance between games is based on shared player ratings, helping you find similar board games instantly.

Use the map to explore clusters of related games, discover hidden gems, and get recommendations based on the titles you already love. It works as a visual alternative to the BGG “Fans Also Like” system.

How to find similar board games

  • Search for any board game in the database
  • Zoom into clusters of related mechanics and themes
  • Identify gateway games and expert-level paths

Based on BoardGameGeek community data

All similarities are computed from real user behaviour and ratings on BoardGameGeek, making this a powerful board game discovery and recommendation engine.
